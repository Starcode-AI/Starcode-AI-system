# Contributing / Mitwirken

## English
Thank you for wanting to contribute to our self-learning local AI project!
* **Pull Requests (PRs):** Fork the repo, create a feature branch (`git checkout -b feature/YourFeature`), and open a PR.
* **Data Policy:** Do not submit copyrighted training data. Any submitted code will be licensed under **GNU GPLv3**.

## Deutsch
Schön, dass Sie zu diesem selbstlernenden, lokalen KI-Projekt beitragen möchten!
* **Pull Requests (PRs):** Forken Sie das Repo, erstellen Sie einen Branch (`git checkout -b feature/IhrFeature`) und öffnen Sie einen PR.
* **Daten-Regel:** Reichen Sie keine urheberrechtlich geschützten Trainingsdaten ein. Eingereichter Code wird unter der **GNU GPLv3** lizenziert.
