# Security Policy / Sicherheitsrichtlinie

## English
If you discover a security vulnerability (e.g., prompt injection flaws, remote code execution risks), please **do not** open a public GitHub Issue.

Please report it directly via email to:
👉 **support@starcode-bom.com**
We will acknowledge your report within 48 hours and work on a local patch.

## Deutsch
Wenn Sie eine Sicherheitslücke finden (z. B. Schwachstellen bei Prompt-Injections oder Risiken zur Code-Ausführung), eröffnen Sie bitte **kein** öffentliches GitHub-Issue.

Bitte melden Sie dies direkt per E-Mail an:
👉 **support@starcode-bom.com**
Wir bestätigen den Erhalt innerhalb von 48 Stunden und arbeiten an einem lokalen Patch.
