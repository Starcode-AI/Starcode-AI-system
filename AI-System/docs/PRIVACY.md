# Privacy notice template

This installation processes account data, session/device metadata, chat content, uploaded files, generated projects, knowledge records, feedback, audit events and security events on infrastructure controlled by the operator.

No content is sent to an external AI provider by LocalAI Control. Web research necessarily sends search terms to the configured private SearXNG service and requests to the selected public source websites. Those sites can observe the research server's IP address and request metadata.

Users can delete conversations and uploaded files, export their data and delete their account. Administrators can access security and audit metadata. Private chat access is not included in the administrator API by default.

The deployment owner must replace this template with the actual controller identity, purpose, legal basis, retention periods, recipients, hosting location, contact information and applicable data-subject rights before offering the service to other people.
